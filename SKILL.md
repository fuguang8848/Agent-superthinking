# ⚔️ 超思考 · 双轨专家系统 v4

> 不同领域的思想家看待同一个问题，结论往往不同——这种差异本身就是洞察的来源。

## 核心架构

```
问题输入
    ↓
┌─────────────────────────────────────────┐
│  路由层（Router）                       │
│                                         │
│  Step 1: 初步判断涉及哪些专家团         │
│  Step 2: 追问用户获取上下文 ← 关键新增 │
│  Step 3: 用户选择轨别和粒度            │
└─────────────────────────────────────────┘
    ↓
被选中的专家/框架并行分析
    ↓
冲突检测 + 共识提炼
    ↓
结构化综合报告
```

**关键改进 v4：**
- 路由层包含**追问机制**，不直接激活专家
- 追问获取上下文 → 分析更命中用户真实情况
- 避免"教科书式回答"——多领域视角但缺乏针对性

---

## 路由层详解

### Step 1: 初步路由

读取问题，提取关键词，初步判断涉及哪些专家团。

### Step 2: 追问机制（新增）

**路由层主动判断：是否需要追问？**

| 判断标准 | 需要追问 | 直接分析 |
|----------|----------|----------|
| 问题类型 | 开放式人生/价值观问题 | 具体操作性决策 |
| 上下文 | 用户没说具体场景 | 用户已提供完整背景 |
| 深度 | 泛泛而问 | 有明确困惑点 |
| 语气 | "人生的意义是什么" | "30岁该不该转行做X" |

**追问原则：**
- 每次最多问 **2 个问题**
- 问具体场景/感受/处境，不问价值观
- 问"是什么让你想到这个"而非"你觉得意义是什么"
- 用户明确说"随便问问"则跳过追问

**追问模板：**

```
[初步判断涉及：哲学团（尼采/庄子/萨特）、宗教团（佛陀）]

在你思考这个问题之前，我想先了解一些背景：

1. 你最近是遇到了什么事，让你想到这个问题？
   （比如：重大决定、挫折、还是纯粹思考人生？）

2. 你现在最困惑的是什么？
   （比如：不知道该选择什么、觉得什么都没意义、还是想找到方向？）

[直接输入背景，或说"随便问问"直接开始分析]
```

### Step 3: 用户选择轨别

```
┌─────────────────────────────────────────┐
│  路由结果（基于初步判断 + 你的背景）：    │
│                                         │
│  人物型：哲学团 + 宗教团                │
│  方法论：存在主义 + 伦理学              │
│                                         │
│  请选择分析轨别：                        │
│  [1] 人物型（推荐）                    │
│  [2] 方法论型                           │
│  [3] 双轨组合                          │
│  [4] 自定义                            │
│                                         │
│  请选择粒度：                            │
│  [A] 完整分析                           │
│  [B] 只要哲学团                        │
│  [C] 指定专家                          │
└─────────────────────────────────────────┘
```

## 完整流程示例

### 示例：人生的意义是什么？

**Step 1 - 初步路由：**
```
判断涉及：哲学团（苏格拉底/尼采/庄子/萨特）、宗教团（佛陀/王阳明）、文学团（加缪）
```

**Step 2 - 追问：**
```
在你思考这个问题之前，我想先了解一些背景：

1. 你最近是遇到了什么事，让你想到这个问题？
2. 你现在最困惑的是什么？
```

**用户回答：** "最近30岁，感觉人生定型了，没什么盼头"

**Step 3 - 选择轨别：**
```
基于你的背景，推荐：
- [3] 双轨组合
- [A] 完整分析

激活专家：尼采（超人哲学）、庄子（逍遥）、萨特（自由责任）、加缪（荒谬反抗）、佛陀（苦）、王阳明（致良知）
```

**最终分析（基于用户背景）：**
```
结合你的具体情况（30岁危机、感觉定型），多视角分析：

尼采会怎么说：...
加缪会怎么说：...
...

结论：针对你的处境，建议...
```

## 与旧版区别

| 维度 | v3 | v4 |
|------|-----|-----|
| 路由层 | 只判断专家团 | 判断 + 追问获取上下文 |
| 回答质量 | 教科书式，泛泛而谈 | 基于用户背景，命中实际 |
| 追问机制 | 无 | 每次最多2个问题 |
| 用户控制 | 选择轨别/粒度 | 选择轨别/粒度 + 提供背景 |

---

## 质量门禁

1. **追问是否必要**：是否真的需要背景才能回答？
2. **追问是否精准**：是否命中用户的真实困惑点？
3. **回答是否命中**：分析是否结合了用户提供的背景？
4. **避免教科书式回答**：不能只列知识点，要结合用户处境
