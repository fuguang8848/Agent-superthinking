# 方法论索引

> 方法论是工具箱。你可以在辩论中调用它们来验证你的观点。
> 想了解某个方法论的详细介绍？去 `experts/methods/<方法论名>/SKILL.md` 看完整文档。

---

## 方法论一览

| 方法论 | 简介 | 何时用 |
|--------|------|--------|
| **美学** | 研究"美"与"艺术体验"的哲学框架。关注形式、情感、审美价值。 | 当问题涉及"美"、"艺术"、"设计品味"、"情感价值"时 |
| **分析哲学** | 用逻辑和语言分析的方法澄清概念。追求清晰、严密、避免语言陷阱。 | 当问题被模糊的概念困扰，需要先澄清"X到底是什么意思"时 |
| **人类学** | 研究人类文化与行为。从田野调查出发，关注文化相对性和整体性。 | 当问题涉及不同文化背景、习俗差异、"本地人怎么看"时 |
| **传播学** | 研究信息在人与人之间的传递过程。关注媒介、效果、噪声、信噪比。 | 当问题涉及"怎么传递信息"、"沟通障碍"、"舆论"时 |
| **控制论** | 研究系统如何通过反馈实现自我调节。核心是"输入-反馈-输出"循环。 | 当问题涉及"反馈机制"、"自我调节"、"系统稳定性"时 |
| **伦理学** | 研究对与错、善与恶。提供道德判断的框架——义务论、美德论、功利论。 | 当问题涉及"应不应该"、"道德还是不道德"、"公平正义"时 |
| **法理学** | 研究法律的本质、正当性、解释规则。追问"什么使一条法律成为法律"。 | 当问题涉及法律基础、权利义务、正当程序、合宪性时 |
| **语言学** | 研究语言的结构与使用。关注语法、语义、语用、方言、社会语言变体。 | 当问题涉及"这个词是什么意思"、"表达是否有歧义"时 |
| **文学批评** | 解读文本的意义与价值。关注叙事结构、视角、修辞、互文性。 | 当问题涉及文本解读、叙事逻辑、隐喻象征时 |
| **管理学** | 研究组织与资源的有效配置。关注效率、激励、治理、战略。 | 当问题涉及"怎么组织"、"怎么分配资源"、"激励机制"时 |
| **叙事学** | 研究叙事的结构与技巧。关注情节、人物、视角、时间线、故事弧。 | 当问题涉及"故事怎么讲"、"情节是否合理"、"人物动机"时 |
| **运筹学** | 把决策问题抽象为数学模型，通过优化、排队、网络流等工具求解。 | 当问题涉及"最优安排"、"资源调度"、"排队效率"、"路径规划"时 |
| **教育学** | 研究教学的规律与学习的效果。关注教学方法、知识传递、能力培养。 | 当问题涉及"怎么教会别人"、"学习规律"、"教育设计"时 |
| **现象学** | 通过"悬置"偏见，还原意识经验的本貌。关注事物如何"显现"给我们。 | 当问题需要追问"这件事的本质是什么"、"我们是怎么体验的"时 |
| **诗学** | 研究诗歌和文学艺术的创造规律。关注形式、韵律、意象、想象力。 | 当问题涉及"怎么表达"、"形式与内容的关系"、"文学创作"时 |
| **政治哲学** | 研究国家、权力、正义的本质。追问"谁应该统治"、"什么使统治合法"。 | 当问题涉及政治制度、权力合法性、公共正义、公民义务时 |
| **修辞学** | 研究说服与表达的技巧。关注论证结构、比喻、情感诉求、可信度。 | 当问题涉及"怎么说服"、"论证是否有漏洞"、"表达是否有误导"时 |
| **社会学** | 研究社会结构与社会行为。关注群体、组织、制度、社会流动。 | 当问题涉及"社会怎么看"、"群体行为"、"制度影响"时 |

## 使用方法

### 作为专家

在辩论中，如果你想用某个方法论验证你的观点，直接说：

```
我用[方法论]检验一下我的论点。
```

然后从该方法论的角度分析你的论点。

完整方法论文档在：`experts/methods/<方法论名>/SKILL.md`

### 作为主持人

如果你觉得某个论点可以用方法论视角检验，可以提示专家：

```
[专家]，要不要用[方法论]的角度看一下这个问题？
```
