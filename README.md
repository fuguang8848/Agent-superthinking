# ⚔️ Agent-Superthinking

> 多维度专家团思考框架。一个问题 → 路由层判断涉及哪些专家团 → 你选择粒度 → 并行分析 → 冲突检测 → 综合报告。

## 核心架构

```
问题输入
    ↓
┌─────────────────────────────────────┐
│  路由层：判断涉及哪些专家团        │
│  → 展示路由结果 + 询问用户选择粒度 │
└─────────────────────────────────────┘
    ↓
用户选择粒度：
├─ [1] 完整分析（所有涉及专家团）
├─ [2] 只要指定专家团
├─ [3] 指定专家（自己选）
└─ [4] 指定具体专家
    ↓
被选中的专家并行分析
    ↓
冲突检测 + 共识提炼
    ↓
结构化综合报告
```

## 使用方式

### OpenClaw Skill 方式

触发词：`思考`、`分析`、`深度分析`、`多视角`、`多维分析`

```
> 帮我分析：AI会不会取代人类？

[路由器展示]
这个问题涉及：哲学团 / 科学团 / 风险审查团

请选择分析粒度：
[1] 完整三团分析
[2] 只要哲学团
[3] 指定专家团
[4] 指定专家
```

### Python 包方式

```bash
pip install agent-superthinking
```

- **原创框架**：Magi 三贤者、元思考、TagMemo、VCP 等

## 设计原则

1. **路由层 = 判断专家团，不是专家**：用户决定分析粒度
2. **按需加载**：只加载被选中的专家 SKILL.md，不全量加载
3. **模块化**：INDEX.md 是唯一真相源，新增专家只需加文件 + 更新 INDEX
4. **可移植**：复制整个目录即可迁移
